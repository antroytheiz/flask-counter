## How to run This App
  * git clone https://github.com/antroytheiz/flask-counter.git
  * cd flask-counter
  * python -m venv env-flask
  * source env-flask/bin/activate
  * pip install flask
  * pip install flask-sqlalchemy
  * pip install Flask-IPInfo
  * python run.py
  * open your browser *localhost:5000*
